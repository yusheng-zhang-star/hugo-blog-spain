## Preguntas frecuentes

**P1: ¿Cuál es el tamaño de archivo recomendado para imágenes de una galería comercial B2B?**
R: Para una web corporativa profesional, cada imagen individual no debería superar los 250 KB cuando se trata de fotografías de proyectos o instalaciones. Si la imagen es una fotografía de alta resolución destinada a una página de "portfolio" detallado, el límite puede ampliarse a 500 KB, siempre que se implemente la carga diferida (lazy loading) en el código de la web.

**P2: ¿Cómo afecta la compresión de imágenes al posicionamiento en Google para negocios locales en Madrid?**
R: El Core Web Vitals de Google incluye métricas de velocidad de carga (LCP) que se ven directamente beneficiadas por la optimización de imágenes. Una web corporativa en Madrid que carga rápidamente tiene mayores probabilidades de posicionarse en la primera página para búsquedas competitivas como "despacho abogados Madrid" o "consultoría tecnológica Madrid".

**P3: ¿Es necesario mantener una copia de las imágenes originales sin comprimir?**
R: Absolutamente. La compresión debe aplicarse siempre sobre una copia de trabajo. Las imágenes originales en alta resolución deben archivarse en un sistema de almacenamiento seguro (NAS local o cloud corporativo) para posibles usos futuros en impresión de material comercial de gran formato.

**P4: ¿Puedo comprimir imágenes que contienen texto incrustado sin que el texto se degrade?**
R: Sí, pero requiere ajustar el algoritmo de compresión. Para imágenes que contienen texto (como infografías o capturas de prensa), recomendamos utilizar el formato WebP con compresión sin pérdida (lossless) o, alternativamente, el formato PNG optimizado. La herramienta permite seleccionar el nivel de calidad para preservar la nitidez de los bordes del texto.

**P5: ¿Cómo puedo medir el impacto de la optimización de imágenes en el rendimiento de mi web corporativa?**
R: Utiliza Google PageSpeed Insights y Google Search Console (sección de "Experiencia en la página") para monitorizar las métricas de Core Web Vitals. Un antes y después de la optimización por lotes revelará mejoras inmediatas en el Largest Contentful Paint (LCP), que es el indicador que mide cuánto tarda en cargarse el elemento de contenido más grande visible en la pantalla.
